# 脚注跳转

这是一个思源笔记插件，用于在不修改笔记数据的情况下，为 Markdown 风格脚注提供类似 Typora 的可点击跳转体验。

支持的写法：

```markdown
正文里引用脚注 [^1]。

[^1]: 这里是脚注内容。
```

启用后：

- 顶部栏会出现脚注按钮，点击后可在菜单中启用或停用插件。
- 菜单中的“编辑块实时刷新”可控制当前编辑块是否自动刷新；关闭后会在光标离开该块时处理。
- `[^1]` 会临时显示为可点击引用。
- 鼠标悬停引用时显示脚注内容预览。
- 点击引用会自动滚动到文档末尾对应的 `[^1]: ...` 定义。
- 插件不会调用更新块数据的 API；关闭开关或卸载插件时会清理临时 DOM。
- 插件使用块级增量刷新，并在中文输入期间暂停处理，避免频繁更新干扰编辑。

## 开发

```bash
npm install
npm run dev
```

开发模式会生成 `index.js` 和 `index.css`。

## 打包

```bash
npm run build
```

打包结果位于 `dist/`，同时生成 `package.zip`。
